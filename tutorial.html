Tagging sounds tutorial

Hi! This is a demo project that shows how you can do sound pattern recognition on audio files the PyBossa framework in Crowdcrafting.org.

The project is really simple. It loads a sound from Freesound and asks you this question: Tag this sound clip

Each sound clip has two handles on top the spectrum. Drag them to select the area of the sound where the idenfied sound is happening (i.e. a dog barking):

When you select an area, an input box will appear and you will be able to write the type of sound that you have identified:

Then, submit the tag and load another sound clip!
